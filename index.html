How It Works

작동 방식

1 사용자가 UI / 음성 등으로 목적지 입력
→
2 카카오맵 API로 길찾기 요청
→
3 경로 수신 후 목적지로 주행 시작
↓
4 실내 주행 시 카메라로 SLAM 지도 데이터 생성 후 실내 지도와 대조
↓
5 목적지 도착
3-1. 실외 주행 시 명령 우선순위
1순위 영상 입력받은 장애물 데이터
2순위 GPS / 보도블럭 유도 OPEN CV
3순위 카카오맵 길찾기 경로
차이점
기존 자율주행 휠체어들은 목적지까지 개입 없이 주행하게 설계된 본 기기와 달리 고정된 장소 또는 장애물 회피 기능밖에 없습니다.
기능 추가 시 경제성이 크게 떨어지는 반면, 본 기기는 추가 장착형이며 고가인 3D Lidar 탑재형 대비 경제성이 개선되었습니다.
