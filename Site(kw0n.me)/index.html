자율주행 휠체어

YOLO26n + Depth Anything V2 + Tmap API를 결합해 실외·실내 목적지까지 자율 주행합니다. 기존 휠체어에 추가 장착 가능한 모듈형 설계입니다.

작동 방식

음성 또는 UI로 목적지를 입력하면, 실외·실내 환경에 맞춰 자동으로 판단하며 주행합니다.

1

목적지 입력

UI 터치 또는 자연어 음성으로 목적지를 입력합니다. AI가 발화 문맥을 분석해 목적지·이동 의도를 자동 추출하므로, 정확한 명령 구조 없이도 평소 말하듯 이야기하면 됩니다.

UI / 음성 인식 · AI 문맥 추출 (클라우드)
2

경로 생성

Tmap API로 실외 보행 경로를 요청하고 수신합니다.

Tmap API · GPS
3

자율 주행 시작

경로를 따라 주행하면서 카메라와 라이다가 실시간으로 장애물을 감지하고 우회합니다.

YOLO26n · Depth Anything V2 · 2D Lidar
4

실내 진입 시 자동 전환

실내 진입이 감지되면 라이다 우선·카메라(SLAM) 보조 모드로 전환해 2D 실내 지도와 대조하여 실내 위치를 파악하고 경로를 추종합니다.

2D Lidar 우선 · SLAM 보조
5

목적지 도착

목적지에 도착하면 음성 및 화면으로 안내합니다.

TTS · 화면 알림

1순위카메라 데이터 (Depth · 장애물 감지 · 우회 판별)
2순위라이다 데이터 (실제 장애물 거리 측정)

2순위카메라 데이터 (SLAM 위치 보정)

기술별 주요 기능

📷

카메라 (Depth + SLAM)

  • YOLO26n 실시간 객체 감지 (장애물·사람·신호 등)
  • Depth Anything V2 — 단안 카메라 거리 추정
  • SLAM — 실내 지도 생성 및 2D 지도와 대조하여 자기 위치 추정
  • 실외 1순위 / 실내 2순위 센서
  • Jetson Orin Nano에서 TRT FP16 최적화
Python / PyTorch · OpenCV · Ultralytics

2D 라이다

  • 실외 장애물 실측 거리 측정

  • 카메라 오감지 보완 역할
  • 저가 2D Lidar로 3D Lidar 대비 경제성 개선
유선연결 · USB

GPS + Tmap API

  • 실외 3순위 명령 소스
Tmap API · LTE/5G 이동통신망

기존 제품과의 차이점

기존 자율주행 이동장치와 세 가지 관점에서 비교합니다.

01 — 경제성

3D Lidar 없이도 동작

고가의 3D Lidar 대신 카메라(Depth·SLAM)와 비교적 저가인 2D Lidar를 조합해 동등한 장애물 회피 성능을 달성했습니다.

02 — 완전 자율주행

목적지까지 개입 없이

기존 제품은 고정된 장소 이동 또는 단순 장애물 회피에 그쳤지만, 본 기기는 장소에 상관없이 처음가보는 장소라도 목적지까지 완전 자율 주행합니다.

03 — 추가 장착형

기존 휠체어에 탈부착

새 휠체어를 구입할 필요 없이 기존 수동 휠체어에 추가 모듈을 장착하는 방식으로 제작 비용을 낮췄습니다.

이건 다른 공모전에 출품할 작품입니다. 교내대회와는 별개의 작품입니다.
